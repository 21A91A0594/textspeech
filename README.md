# Text to Speech Converter App

This is a web application that converts text input into speech using HTML, CSS, and JavaScript.

## Features

- **Text to Speech Conversion**: Enter text and have it converted into spoken words.
- **Language Selection**: Choose the language for speech output (if applicable).
- **Speech Controls**: Play, pause, and stop buttons for controlling speech playback.
- **Responsive Design**: The app is designed to work on different screen sizes.

## Technologies Used

- HTML
- CSS
- JavaScript
  Web Speech API (if used for text-to-speech functionality)
